# 🚗 Car Price Prediction & Interactive Analytics Platform

## 📌 Overview

This project is a full end-to-end **Data Analysis + Machine Learning pipeline** for predicting car prices.
It goes beyond traditional prediction models by offering **interactive analytics, dynamic visualizations, and customizable target selection**.

The system allows users to:

* Explore car datasets visually 📊
* Extract actionable insights 💡
* Train machine learning models ⚙️
* Predict prices dynamically 🎯

---

## 🚀 Key Features

### 📊 1. Data Analysis & Visualization

* Advanced exploratory data analysis (EDA)
* Interactive visualizations (dynamic plots & charts)
* Correlation analysis between features
* Distribution analysis for numerical features

### 💡 2. Insights Generation

* Automatic pattern detection
* Feature importance identification
* Trend analysis across car attributes
* Business-oriented insights

---

### 🤖 3. Machine Learning Pipeline

* Data preprocessing (cleaning, encoding, scaling)
* Multiple model training:

  * Linear Regression
  * Decision Tree
  * Random Forest
* Model evaluation using:

  * Accuracy metrics
  * Error analysis

---

### 🎯 4. Dynamic Target Control

One of the most powerful features of this project:

* Ability to **select or change the target variable dynamically**
* Re-train models based on selected target
* Compare model performance across different targets

---

### 📈 5. Prediction System

* Input custom car features
* Real-time price prediction
* Model-based decision output

---
